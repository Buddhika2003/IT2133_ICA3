# Login & Signup Web Application (PHP)

This project demonstrates a simple LOGIN AND SIGNUP PAGE built using PHP and MySQL.  
It includes two branches to show both a vulnerable and a solution(fixed) version of the same web app.

# Project Structure

•main branch → Final merged version of the app  
•vulnerable branch → Contains insecure version (with CSRF vulnerability)  
•solution branch → Contains fixed version with CSRF protection

# Features
•User registration and login  
•Session management  
•Email change option (with and without CSRF protection)  
•Demonstrates secure and insecure coding practices  

---

# Learning Outcome

•Understand how web vulnerabilities like CSRF work  
•Learn how to fix vulnerabilities using proper validation and tokens  
•Practice using Git and GitHub for version control and collaboration
